subString
Type: SystemString
string - Expression to be added to subscription list. Expression will provide a header value to be used as a filter. If specified the only messages that will be received on the current connection will be those Satisfying the expression or those msgSelectors specified in addSubscription. If no subscriptions are submitted then this system will receive all messages published to the topic
Remarks
Host will only publish messages matching one of the selectors. If no addSubscriptions are submitted then all messages will be received.
